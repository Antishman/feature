🎓 Arsi University Job Portal Laravel + Tailwind

A modern web platform connecting jobseekers, employers, and admins at Arsi University. Built with ❤️ using Laravel and Tailwind CSS.

🚀 Features Overview

🧑‍💼 Jobseekers

  • 🔍 Browse Jobs – Filter by title, category, location, date
  • 💾 Save Jobs – Bookmark for later
  • 📨 Apply – Apply once per job, no duplicates
  • 📝 Resume Builder – Fillable, editable, PDF-ready
  • 🔔 Notifications – Track job updates, applications
  • 📢 Announcements – Get news from the university
  • 👤 Dashboard – Personalized, clean, responsive

🏢 Employers

  • 📄 Post Jobs – Describe, tag, and submit jobs (pending until approved)
  • 🗂 Manage Jobs – Edit, delete, or check applications
  • 📥 Applications – View applicants and resumes
  • 👤 Dashboard – Track jobs and performance

🛠️ Admin

  • 📢 Manage Announcements – Add or edit university news
  • 📊 Site Analytics – Users, jobs, applications
  • 👥 Manage Users – Role view, delete access
  • 📌 Manage Jobs – Approve, reject, or remove jobs

🧱 Tech Stack

  • Laravel 10 – Back-end framework
  • Tailwind CSS – Front-end utility-first styling
  • Blade Templates – For dynamic pages
  • MySQL – Data storage
  • Authentication – Laravel Breeze for login/register

📎 Setup Instructions

git clone https://github.com/antishman/arsijobportal.git
cd arsi-job-portal

composer install
npm install && npm run dev

cp .env.example .env
php artisan key:generate
php artisan migrate --seed

php artisan serve
